# KbuO Code site

Static developer website for KbuO Code mobile games.

This site is intended for:

- developer profile and game links
- Google AdMob app-ads.txt verification
- privacy policy pages for published games

## Cloudflare Pages

Use these settings:

- Framework preset: None
- Build command: leave empty
- Output directory: /

## Important URLs

After deployment, verify:

- /app-ads.txt
- /privacy/idle-merge-farm.html
- /privacy/bomb-defuse.html
